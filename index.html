Математические пакеты

Wolfram Alpha
Wolfram Alpha
Описание:

Математический пакет, который поможет решить задачи на любой вкус и цвет (интерфейс на английском языке). В бесплатной версии система выдает только ответ, а подписка позволяет просмотреть полное решение.

Ссылка:
https://www.wolframalpha.com/
Пригодится для вычисления:
  • арифметических выражений;
  • пределов;
  • производных;
  • интегралов,

а также для решения многих других задач

Desmos
Desmos
Описание:

Графический калькулятор, который позволяет строить графики функций на одной плоскости

Ссылка:
https://www.desmos.com/
Пригодится для построения:
  • графиков функций одной переменной;
  • линий уровня (уравнений f(x) = C)
Math Semestr
Math Semestr
Описание:

Математический пакет с широким функционалом. В отличие от WolframAlpha, полное решение доступно бесплатно, а интерфейс на русском языке.

Ссылка:
https://math.semestr.ru/
Пригодится для решения задач:
  • по математическому анализу (вычисление пределов, производных, интегралов и т.д.);
  • по линейной алгебре (определитель матрицы, операции над матрицами и т.д.)
